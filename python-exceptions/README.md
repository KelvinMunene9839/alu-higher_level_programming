# python-exceptions

Python fundamentals: exception handling with `try`/`except`/`finally`,
built-in exception types, and raising exceptions with `raise`.

## Tasks

| File | Description |
| --- | --- |
| `0-safe_print_list.py` | Prints x elements of a list, catching `IndexError` |
| `1-safe_print_integer.py` | Prints a value as an integer, catching invalid types |
| `2-safe_print_list_integers.py` | Prints the integers among the first x elements of a list |
| `3-safe_print_division.py` | Divides 2 integers and prints the result in a `finally` block |
| `4-list_division.py` | Divides 2 lists element by element, handling multiple exception types |
| `5-raise_exception.py` | Raises a `TypeError` exception |
| `6-raise_exception_msg.py` | Raises a `NameError` exception with a message |

## Requirements

- Ubuntu 20.04 LTS, Python 3.8.5
- Style checked with `pycodestyle` 2.7.*
- Every file starts with `#!/usr/bin/python3` and is executable

# ALU Higher Level Programming

This repository contains projects completed as part of the Higher Level
Programming curriculum, covering Python and Shell scripting fundamentals.

## Projects

### python-hello_world

An introduction to Python: the interpreter, `print`, strings, indexing and
slicing, f-strings, and the `pycodestyle` style guide. Also includes two
Bash scripts for running Python files and inline Python code.

### python-if_else_loops_functions

Conditional statements, `while`/`for` loops, loop control (`break`,
`continue`, loop `else` clauses, `pass`), the `range` function, functions,
variable scope, and arithmetic operators.

### python-more_data_structures

Sets, set methods, dictionaries, dictionary methods, lambda functions, and
the `map`, `reduce`, and `filter` functions.

### python-exceptions

Exception handling with `try`/`except`/`finally`, built-in exception types,
and raising exceptions with `raise`.

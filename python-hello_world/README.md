# python-hello_world

A first look at Python: running scripts and inline code from Bash, the
`print` function, string indexing/slicing, f-strings, and writing
`pycodestyle`-compliant code.

## Tasks

| File | Description |
| --- | --- |
| `0-run` | Bash script that runs the Python file named in `$PYFILE` |
| `1-run_inline` | Bash script that runs the Python code stored in `$PYCODE` |
| `2-print.py` | Prints a quoted sentence with `print` |
| `3-print_number.py` | Prints an integer using an f-string |
| `4-print_float.py` | Prints a float rounded to 2 digits using an f-string |
| `5-print_string.py` | Prints a string repeated 3 times and its first 9 characters |
| `6-concat.py` | Builds a sentence from two strings |
| `7-edges.py` | Slices a string into its first 3, last 2, and middle characters |
| `8-concat_edges.py` | Builds a new sentence purely from slices of an existing string |
| `9-easter_egg.py` | Prints the Zen of Python |

## Requirements

- Ubuntu 20.04 LTS, Python 3.8.5
- Every Python file starts with `#!/usr/bin/python3`, every Bash script
  with `#!/bin/bash`
- All files are executable and end with a newline
- Python code follows `pycodestyle` (2.7.*)
